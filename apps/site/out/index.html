Built for developers who care about control

No magic. No hidden API calls. Every action is transparent, approvable, and auditable.

Security-First

Default deny for writes and commands. Workspace sandboxing. Approval gates for every risky action. Secrets firewall built in.

Any LLM Provider

OpenAI-compatible, Anthropic native, or any custom HTTP API. Pluggable provider adapters. No vendor lock-in.

Diff-Based Writes

Every file change shows a clear diff for review before it touches your code. No surprises.

Full Audit Trail

Every run logged with tasks, plans, approvals, tool calls, and results. Replay and inspect any session.

Local Execution

Everything runs on your machine. Your code never leaves your workspace. Only LLM API calls go to the network.

Interactive Chat

Full conversational agent in your terminal. Plan, discuss, review, and approve actions in real time.

Ready to get started?

Install Rivet in seconds and start running agentic workflows from your terminal.

$ npm i -g @pulsesparkai/rivet

$ rivet init

$ rivet run "add tests to utils.ts"
